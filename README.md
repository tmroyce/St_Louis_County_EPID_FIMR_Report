# St_Louis_County_EPID_FIMR_Report

The repository is organized into the following key folders and files:

| **Item**                       | **Description**                                                                                                                           |
| ------------------------------ | ----------------------------------------------------------------------------------------------------------------------------------------- |
| `data/`                        | Contains the **datasets** used for the analysis                                                                                          |
| `export/`                      | Stores **exported figures and tables** generated by the R code, which are then included in the final report and presentation             |
| `R_code_stlco_fimr_report.Rmd` | The primary **R Markdown file** containing the analysis code, visualizations, and narrative used to generate the final report and slides |
| `Report_stico_fimr.pdf`        | The final rendered **PDF report** detailing the findings of the FIMR project                                                             |
| `Slides_stlco_fimr.pdf`        | A **PDF version of the presentation slides**                                                                                             |
| `Slides_stlco_fimr.pptx`       | The **PowerPoint (PPTX) version of the presentation slides**                                                                             |
